# La Isla 3D

Juego web 3D del festival La Isla.

## Jugar

Abre `index.html` desde el repositorio o publica GitHub Pages desde `main / root`.

URL esperada de Pages cuando esté activado:

`https://uptobe8.github.io/laisla/`

## Incluye

- 9 personajes humanos 3D: Lala, Lety, Isa, Patri, Luis, Fran, Clara, Jesucristo y Aitor.
- Mujeres y hombres asignados correctamente.
- Sin Xbot ni maniquí geométrico.
- Música generada en Web Audio.
- Barra con camareros, pista con gente bailando, papas revolconas, water La Moraleja con llave, carpeta rosa, pistola de agua, tienda sin palos, colchón pinchado, barca, DJ Bueno / DJ Malo, hamaca, fuente y pole dance.
